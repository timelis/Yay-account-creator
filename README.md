# これは何？
[Yay](https://yay.space)のアカウントを作成するプログラムです

# 使い方
1. このリポジトリをダウンロード
2. ダウンロードし、解凍したフォルダで以下のコマンドを実行
```
pip3 install -r requirements.txt
```
3. 実行
```
python3 main.py
```
